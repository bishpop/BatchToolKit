# ⭐ Batch Tool Kit
A project that converts PNG/DDS/GIF. It allows to convert .png into .dds; .gif into .png; .dds into .png.  
Includes Batch rename.

## 📌How to use
1. Download the zip here: [Release](https://github.com/bishpop/BatchToolKit/releases/tag/v1.0)
2. Extract the BatchToolKit.zip
3. execute the .exe
4. select what you would like to change.
